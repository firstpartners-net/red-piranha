Sample Under Development.
Notes: Read information from a complex Excel Spreadsheet and summarize as json/csv.

The sample demonstrates preprocessing to add Named ranges to an Excel file - often needed for historical documents and used to identify the Cells we want to extact and pass into the Rule Engine.

The sample directory contains a prepropssing script called InjectNamedRange.Groovy. There is a setting in application.properties that causes files with this name to be excecuted after the Spreadsheet is loaded but before the business rules are executed.

This script is passed a copy of the incoming spreadsheet which allows it to be manipulated. In this cases the script adds Excel Named Ranges tagging the areas of the sheet we are most interested in.
